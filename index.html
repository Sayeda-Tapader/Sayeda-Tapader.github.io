Sayeda Tapader

In 2021 I joined City, University of London as a student in their BSc Computer Science course. I am creating this portfolio as a part of my Java module

Java Game

My game is a platform game that I developed in my first year studying Computer Science at City, University of London. It leverages a simple didactic Physics Library that wraps around JBox2D/Box2D, which allows programmers to create and simulate 2D world populated by bodies with diverse shapes and appearances, create soundscapes and specify custom keyboard and mouse interactions. The game demonstrates the main character, the player navigate around, while staying far from the enemy and collecting coins. The character can jump up by pressing J and go left and right by pressing L and R, respectively. It can also shoot bullets.
